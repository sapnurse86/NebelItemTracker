# Nebel Item Tracker

**Nebel Item Tracker** is a World of Warcraft Classic addon designed to help guilds, raids, and groups coordinate item needs efficiently.  
It allows players to track required items, share those needs with others, and instantly see who needs what directly in item tooltips.

Lightweight, Classic-compatible, and built with a strong focus on guild cooperation and clear communication.

---

## Project Summary

**Nebel Item Tracker** helps World of Warcraft Classic guilds and groups track needed items, synchronize requirements with party, raid, and guild members, and display who needs what directly in item tooltips.

---

## Key Features (English)

- **Track Needed Items** – Add items by name, item ID, item link, or drag & drop.
- **Guild & Group Sharing** – Automatically share item needs with party, raid, and guild members.
- **Tooltip Integration** – Instantly see who needs an item and how many directly in item tooltips.
- **Drag & Drop Workflow** – Drop an item into the list and set the required amount via a dialog.
- **Classic-Compatible** – Designed specifically for World of Warcraft Classic (Era / Season of Discovery / Wrath).
- **Lightweight & Non-Intrusive** – Clean UI with no noticeable performance impact.
- **Persistent Storage** – Item needs are saved between sessions.
- **Easy Access** – Simple `/nebel` command to open or close the addon.

---

## Hauptfunktionen (Deutsch)

- **Benötigte Items verwalten** – Items per Name, ID, Item-Link oder Drag & Drop hinzufügen.
- **Gilden- & Gruppen-Synchronisation** – Teilt Item-Bedarf automatisch mit Gruppe, Raid und Gilde.
- **Tooltip-Integration** – Zeigt direkt im Tooltip, wer ein Item benötigt und in welcher Menge.

---

## Usage

- Open the addon with: `/nebel`
- Add items manually using the input field and **Add** button
- Or drag an item from your bags into the window to add it
- Hover over items in your inventory to see which players need them

---

## Installation

### CurseForge
1. Download the addon from CurseForge
2. Extract it into your `World of Warcraft/_classic_/Interface/AddOns` folder
3. Ensure the folder name is `Nebel`
4. Restart the game or run `/reload`

### Manual Installation
1. Download the ZIP from this repository
2. Extract the `Nebel` folder into `World of Warcraft/_classic_/Interface/AddOns/`
3. Restart the game or reload the UI
